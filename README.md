# 📜 Elder Edda

Добро пожаловать в **Elder Edda** — современное веб-приложение для исследования древнескандинавской поэзии и мифологии, основанное на "Старшей Эдде". Проект сочетает современную веб-разработку и уважение к культурному наследию, обеспечивая быстрый и удобный пользовательский опыт.

---

## 🚀 Особенности

- **App Router (Next.js)** — серверный рендеринг и статическая генерация для высокой производительности и SEO.
- **Мультиязычность** — i18n через `next-intl` (RU/EN).
- **Темизация** — переключение темы через `next-themes`.
- **Адаптивный UI** — Tailwind CSS + SCSS для современного интерфейса.
- **Типизация** — TypeScript во всем приложении.
- **Нечеткий поиск** — `Fuse.js` для поиска по контенту.
- **Бесконечная прокрутка** — плавная подгрузка данных.

---

## 🛠️ Используемые технологии

- **Next.js (App Router)** — фреймворк React для SSR и SSG
- **React** — библиотека для построения UI
- **TypeScript** — строгая типизация
- **next-intl** — локализация интерфейса
- **next-themes** — управление темой
- **Tailwind CSS** — утилитарный CSS-фреймворк
- **SCSS (Sass)** — расширенный CSS
- **Radix UI / shadcn-ui** — базовые UI-компоненты
- **Fuse.js** — нечеткий поиск
- **React Infinite Scroll Component** — бесконечная прокрутка
- **ESLint** — анализатор кода
- **PostCSS** — трансформация CSS

## 📂 Структура проекта

```
elderedda/
├── src/app/                # App Router (страницы и layout)
│   ├── [locale]/            # локализованные маршруты
│   ├── components/          # компоненты приложения
│   ├── styles/              # SCSS и общие стили
│   └── utils/               # утилиты
├── components/ui/           # UI-компоненты (Radix/shadcn)
├── messages/                # локализации (en/ru)
├── public/                  # статические файлы
├── next.config.js           # конфигурация Next.js
├── tailwind.config.ts       # конфигурация Tailwind
└── package.json             # зависимости проекта
```

## 📚 Использование

### Локальный запуск

```bash
npm install
npm run dev
```

Откройте `http://localhost:3000` в браузере.

### Сборка и запуск в продакшене

```bash
npm run build
npm run start
```

### Линтинг

```bash
npm run lint
```
